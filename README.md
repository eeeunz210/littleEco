# litteEco
자원 순환 정보 공유 플랫폼 사이트 '리틀에코'<br>
2023.12.10 - 2024.01.11 <br>
- 제로 웨이스트를 주제로 자원순환 정보를 공유하고 제품을 제공하는 브랜드를 공유하는 플랫폼 <br>
- 자원순환의 실천과 행동을 독려하기 위한 정보 공유 게시판구현
- 취향에 따른 정보를 메인 페이지에 직관적으로 정보 제공<br>
